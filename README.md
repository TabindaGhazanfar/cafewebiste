# ☕ Coffee 3D UI Website

A modern, fully responsive coffee shop website built using **React, Vite, and Tailwind CSS**.  
This project focuses on beautiful UI design, smooth animations, and a great user experience.



## ✨ Features

- ⚡ Built with React + Vite for fast performance  
- 🎨 Styled using Tailwind CSS  
- 🧭 Smooth scroll navigation between sections  
- 🎬 Smooth animations for UI elements  
- 🧩 SVG/shape dividers for modern section design  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- ☕ Clean and modern coffee shop UI  

---

## 🛠️ Tech Stack

- React  
- Vite  
- Tailwind CSS  
- JavaScript (ES6+)  
- CSS Animations  



## 🎬 Animations & UI

- Smooth scroll transitions  
- Fade-in & slide animations  
- Hover effects on buttons and cards  
- Section-based layout with dividers  



## 🚀 Purpose

This project was built to practice modern frontend development, UI design, and animation handling using React ecosystem.



## 📦 Installation

npm install
npm run dev
